## TouchDesigner Visual Experiment — From Photography to Real-time Visual Systems
<br>
<br>
This is an experimental visual project based on TouchDesigner, generated from a composition of 16 photographs taken from my daily photography practice.  
The project does not aim to create “visual effects,” but rather serves as an exploration of visual structure, rhythm, and perception, starting from personal image-based experience.

I treat photographic works as “visual materials,” deconstructing and recombining them through TouchDesigner’s node system (TOP / CHOP). By introducing trigger logic, parameter mapping, and real-time control, static images are transformed into a visual system with temporality and a sense of feedback.
<br>
> In this process, I focused primarily on:
> <br>
    1. How to translate visual intuition into adjustable and reusable system structures  
> <br>
		2. The behavior of image features (brightness, edges, hierarchy) in real-time processing  
> <br>
    3. How visual change can function as a feedback mechanism, rather than mere decoration  
> <br>
    4. The similarities between non-linear node-based workflows and data flow & state management in frontend / interaction design

This project is both a form of personal visual expression and an experimental practice in integrating UI / UX / frontend development with creative technology.  
At the same time, it reflects my ongoing interest in Computer Vision and real-time visual systems—how images can be “understood,” not just displayed.
<br>
All technical explorations stem from my love for photography and visual expression itself.
<br>
<br>
<br>
<br>

这是一个基于 TouchDesigner 的实验性视觉项目，由我日常摄影的 16 张照片 组合生成。
项目并不以“特效”为目标，而是一次从 个人影像经验出发，对视觉结构、节奏与感知的探索。

我将摄影作品视为“视觉素材”，通过 TouchDesigner 的 节点系统（TOP / CHOP） 将其拆解、重组，并引入触发逻辑、参数映射和实时控制，让静态影像转化为一个具有时间性与反馈感的视觉系统。
<br>
> 在这个过程中，我重点关注了：
> <br>
><br>
    1.如何将视觉直觉转化为可调节、可复用的系统结构
> <br>
><br>
		2.图像特征（亮度、边缘、层级）在实时处理中的表现
> <br>
><br>
    3.视觉变化如何作为一种 反馈机制（feedback），而不仅是装饰
> <br>
><br>
    4.非线性节点流程与前端/交互设计中 数据流与状态管理 的相似性

这个项目既是一次个人视觉表达，也是一种对 UI / UX / 前端与创意技术结合方式 的实践尝试。
同时，它也反映了我对 Computer Vision 与实时视觉系统 的持续兴趣——如何让图像“被理解”，而不仅仅是被展示。
<br>
所有技术探索都源于我对摄影与视觉本身的热爱。
<br>
